数组

Array

O(1) 访问

数据结构特点

数组是最基本的数据结构，存储相同类型元素的连续内存空间。支持随机访问，但大小固定，插入删除操作效率较低。

操作复杂度：

  • • 随机访问：O(1)
  • • 插入/删除：O(n)
  • • 查找元素：O(n)
  • • 内存连续，缓存友好

Python实现

# Python列表就是动态数组
arr = [1, 2, 3, 4, 5]
# 访问元素
print(arr[2]) # 输出: 3
# 插入元素
arr.insert(2, 10) # 在索引2插入10
# 删除元素
arr.pop(2) # 删除索引2的元素
# 切片操作
sub_arr = arr[1:4] # 获取子数组

栈

Stack

O(1) 操作

数据结构特点

栈是一种后进先出（LIFO）的数据结构，只能在栈顶进行插入和删除操作。常用于函数调用、表达式求值、括号匹配等场景。

操作复杂度：

  • • 入栈：O(1)
  • • 出栈：O(1)
  • • 查看栈顶：O(1)
  • • LIFO原则

Python实现

class Stack: def __init__(self): self.items = [] def push(self, item): self.items.append(item) def pop(self): if not self.is_empty(): return self.items.pop() return None def peek(self): if not self.is_empty(): return self.items[-1] return None def is_empty(self): return len(self.items) == 0 def size(self): return len(self.items) # 使用示例 stack = Stack() stack.push(1) stack.push(2) stack.push(3) print(stack.pop()) # 输出: 3

队列

Queue

O(1) 操作

数据结构特点

队列是一种先进先出（FIFO）的数据结构，在队尾插入元素，在队头删除元素。常用于任务调度、广度优先搜索、消息队列等场景。

操作复杂度：

  • • 入队：O(1)
  • • 出队：O(1)
  • • 查看队头：O(1)
  • • FIFO原则

Python实现

from collections import deque class Queue: def __init__(self): self.items = deque() def enqueue(self, item): self.items.append(item) def dequeue(self): if not self.is_empty(): return self.items.popleft() return None def front(self): if not self.is_empty(): return self.items[0] return None def is_empty(self): return len(self.items) == 0 def size(self): return len(self.items) # 使用示例 queue = Queue() queue.enqueue(1) queue.enqueue(2) queue.enqueue(3) print(queue.dequeue()) # 输出: 1

二叉树

Binary Tree

O(log n)

数据结构特点

二叉树是每个节点最多有两个子节点的树形结构。二叉搜索树（BST）是一种特殊的二叉树，左子树所有节点值小于根节点，右子树所有节点值大于根节点。

操作复杂度：

  • • 查找：O(log n)（平衡树）
  • • 插入：O(log n)（平衡树）
  • • 删除：O(log n)（平衡树）
  • • 层次结构

Python实现

class TreeNode: def __init__(self, val=0, left=None, right=None): self.val = val self.left = left self.right = right class BinarySearchTree: def __init__(self): self.root = None def insert(self, val): self.root = self._insert(self.root, val) def _insert(self, node, val): if not node: return TreeNode(val) if val < node.val: node.left = self._insert(node.left, val) else: node.right = self._insert(node.right, val) return node def search(self, val): return self._search(self.root, val) def _search(self, node, val): if not node or node.val == val: return node if val < node.val: return self._search(node.left, val) return self._search(node.right, val)

哈希表

Hash Table

O(1) 平均

数据结构特点

哈希表通过哈希函数将键映射到数组索引，实现快速查找。Python中的字典（dict）和集合（set）就是基于哈希表实现的。

操作复杂度：

  • • 查找：O(1) 平均
  • • 插入：O(1) 平均
  • • 删除：O(1) 平均
  • • 哈希冲突处理

Python实现

# Python字典就是哈希表 hash_table = {} # 插入键值对 hash_table['name'] = 'Alice' hash_table['age'] = 25 hash_table['city'] = 'New York' # 查找 print(hash_table['name']) # 输出: Alice # 更新 hash_table['age'] = 26 # 删除 del hash_table['city'] # 检查键是否存在 if 'name' in hash_table: print("Name exists") # 获取所有键值对 for key, value in hash_table.items(): print(f"{key}: {value}") # 使用defaultdict避免KeyError from collections import defaultdict count = defaultdict(int) count['a'] += 1 # 自动初始化为0
