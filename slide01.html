变量与 数据类型

掌握Python中的变量概念和核心数据类型，为算法编程打下基础

# 变量赋值
x = 10
name = "Alice"
pi = 3.14159
# 数据类型
print(type(x)) #
print(type(name)) #
print(type(pi)) #

基本数据类型

Basic Types

整数 (int)

任意大小的整数，支持大数运算

x = 12345
y = -67890
big_num = 12345678901234567890

浮点数 (float)

双精度浮点数，科学计数法

pi = 3.14159
e = 2.71828
scientific = 1.23e-4

布尔值 (bool)

True 或 False，用于逻辑判断

is_valid = True
is_empty = False
result = 5 > 3 # True

容器类型

Container Types

列表 (list)

可变序列，支持增删改查

numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", 3.14, True]
empty = []

元组 (tuple)

不可变序列，性能更好

coordinates = (10, 20)
colors = ("red", "green", "blue")
single = (42,) # 单元素元组

字典 (dict)

键值对映射，快速查找

student = {"name": "Alice", "age": 20}
scores = {"math": 95, "english": 88}

字符串

String

字符串创建

s1 = "Hello World"
s2 = 'Python Programming'
s3 = """多行字符串"""

常用操作

len(s) # 字符串长度
s.upper() # 转大写
s.lower() # 转小写
s.split() # 分割字符串

字符串切片

s = "Hello World"
s[0:5] # "Hello"
s[6:] # "World"
s[::-1] # 反转字符串

类型转换

Type Conversion

数字转换

int("123") # 123
float("3.14") # 3.14
str(42) # "42"

容器转换

list("abc") # ['a', 'b', 'c']
tuple([1,2,3]) # (1, 2, 3)
set([1,2,2,3]) # {1, 2, 3}

# 快速读取多个数字
nums = list(map(int, input().split()))
