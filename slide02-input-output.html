格式化输出

Formatted Output

f-string (推荐)

name = "Alice"
age = 20
print(f"姓名: {name}, 年龄: {age}")
print(f"π的值: {pi:.2f}")

format()方法

print("姓名: {}, 年龄: {}".format(name, age))
print("π的值: {:.2f}".format(pi))

百分号格式化

print("姓名: %s, 年龄: %d" % (name, age))
print("π的值: %.2f" % pi)
