# 比较运算示例
x = 10
y = 20
print(x == y) # False
print(x != y) # True
print(x < y) # True
print(x > y) # False
print(x <= y) # True
print(x >= y) # False

# 逻辑运算示例
a = True
b = False
print(a and b) # False
print(a or b) # True
print(not a) # False
print(not b) # True
短路特性

Python中的逻辑运算符具有短路特性，可以提高程序效率

位运算符

Bitwise Operators

&
按位与
|
按位或
^
按位异或
~
按位取反
<<
左移
>>
右移
# 位运算示例
a = 12 # 1100
b = 10 # 1010
print(a & b) # 8 (1000)
print(a | b) # 14 (1110)
print(a ^ b) # 6 (0110)
print(a << 1) # 24 (11000)
print(a >> 1) # 6 (0110)
熟练运用各类运算符可以大大简化算法实现
