实例：判断奇偶

num = int(input("请输入数字: "))
if num % 2 == 0:
print(f"{num} 是偶数")
else:
print(f"{num} 是奇数")
注意事项

多分支结构

Multiple Branches

if-elif-else

score = int(input("请输入分数: "))
if score >= 90:
grade = "A"
elif score >= 80:
grade = "B"
elif score >= 70:
grade = "C"
else:
grade = "D"
print(f"等级: {grade}")

嵌套if语句

num = int(input())
if num > 0:
if num % 2 == 0:
print("正偶数")
else:
print("正奇数")
elif num < 0:
print("负数")
else:
print("零")
最佳实践

尽量使用elif而不是多个if，这样逻辑更清晰，效率更高

条件表达式

Conditional Expression

三元运算符

# 语法: value_if_true if condition else value_if_false
age = int(input("请输入年龄: "))
status = "成年" if age >= 18 else "未成年"
print(status)

链式条件

x = int(input())
result = ("正数" if x > 0 else
"负数" if x < 0 else "零")
print(result)
使用场景

适合简单的条件赋值，使代码更简洁，但复杂逻辑仍建议使用if-else

实用案例

Practical Examples

判断闰年

year = int(input("请输入年份: "))
if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
print(f"{year} 是闰年")
else:
print(f"{year} 不是闰年")

求最大值

a, b, c = map(int, input().split())
max_val = a
if b > max_val:
max_val = b
if c > max_val:
max_val = c
print(f"最大值: {max_val}")
