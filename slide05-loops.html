循环 Loops

掌握for循环和while循环，实现重复操作的自动化处理

# for循环遍历
for i in range(5):
print(i) # 0, 1, 2, 3, 4
# while循环
i = 0
while i < 5:
print(i)
i += 1

for循环

For Loop

range()函数

# range(stop)
for i in range(5):
print(i) # 0,1,2,3,4
# range(start, stop)
for i in range(2, 6):
print(i) # 2,3,4,5
# range(start, stop, step)
for i in range(0, 10, 2):
print(i) # 0,2,4,6,8

遍历序列

# 遍历列表
fruits = ["apple", "banana", "orange"]
for fruit in fruits:
print(fruit)
# 遍历字符串
for char in "Python":
print(char)
使用技巧

使用enumerate()可以同时获取索引和值：for i, val in enumerate(list)

猜数字游戏

secret = 42
guess = int(input("猜数字: "))
while guess != secret:
if guess < secret:
print("太小了")
else:
print("太大了")
guess = int(input("再试一次: "))
print("猜对了!")
注意事项

确保循环条件最终会变为False，否则会造成无限循环

循环控制

Loop Control

break语句

# 找到第一个质数
for num in range(2, 100):
for i in range(2, int(num**0.5) + 1):
if num % i == 0:
break
else:
print(f"找到质数: {num}")
break

continue语句

# 跳过偶数
for i in range(1, 11):
if i % 2 == 0:
continue
print(i) # 只打印奇数
使用区别

break完全终止循环，continue跳过当前迭代继续下一次

经典算法

Classic Algorithms

阶乘计算

n = int(input("请输入数字: "))
factorial = 1
for i in range(1, n + 1):
factorial *= i
print(f"{n}! = {factorial}")

斐波那契数列

n = int(input())
a, b = 0, 1
for i in range(n):
print(a, end=' ')
a, b = b, a + b
